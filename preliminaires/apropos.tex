Il va de soi que ce livre a été composé avec \XeLaTeX. Outre les packages dont il traite, j'ai utilisé afin de le composer les packages \package{multicol} pour gérer finement le double colonage de l'index ; \package{ltxdockit} et \package{minted} pour les citations de code ; \package{framed} pour les boîtes colorées, et enfin, \package{bidi} pour les commandes \XeLaTeX et \XeTeX.
